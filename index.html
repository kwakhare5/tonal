Shadow DOM Isolation

Every Tonal UI component is fully wrapped inside isolated Shadow Roots. Host styles never bleed in, and extension styles never disrupt the page's styling rules.

Zero Dependencies

No build step. No npm. No bundlers. Written purely in Vanilla Javascript and CSS, optimizing performance, page loads, and extension package size.

Secure Proxy Architecture

No API keys are stored in the client-side content scripts. All traffic is securely encrypted and routed via Service Workers to a serverless Cloudflare Workers proxy.

Install on Chrome

Tonal is developed with standard manifest-driven modular scripts. You can run the extension locally in seconds by loading it unpacked.

1

Clone / Download Code

Download the codebase files and keep the directory structured as is.

2

Open Extensions Panel

Navigate to chrome://extensions/ in your Chrome web browser address bar.

3

Toggle Developer Mode

Switch on the Developer mode toggle located in the upper-right corner of the window.

4

Load Unpacked Extension

Click the Load unpacked button on the top-left, and select the folder containing manifest.json.

manifest.json
{
  "manifest_version": 3,
  "name": "Tonal",
  "version": "5.0.0",
  "permissions": [
    "storage"
  ],
  "host_permissions": [
    "https://mail.google.com/*",
    "https://*.slack.com/*",
    "https://*.linkedin.com/*"
  ]
}
