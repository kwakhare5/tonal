Tonal Extension Design System Catalog

Official component catalog, design tokens, markup templates, and interactive state triggers for the Tonal Chrome Extension. All UI components render within an isolated Shadow DOM container in production.

1. Color & Typography Tokens

Standard styling constants used inside tonal.css. Ensures a consistent theme that matches the main Tonal brand.

2. Floating Pill Component States

The floating capsule pill that anchors inside inputs. It transitions dynamically across five structural states based on current operation.

1. Rest State t-pill--rest
Default low-profile mode shown when input is active but the user is typing without selecting a tone.
<div class="t-pill t-pill--rest">
  <div class="pill-icon">
    <img src="icons/icon128.png" />
  </div>
</div>
2. Expanded State t-pill--expanded
Work
Expanded mode displaying the currently selected tone option along with a chevron toggle dropdown.
<div class="t-pill t-pill--expanded">
  <div class="pill-icon">
    <img src="icons/icon128.png" />
  </div>
  <span class="pill-text">Work</span>
  <div class="pill-chev-wrap"><svg>...</svg></div>
</div>
3. Converting State t-pill--loading
Converting
Active loading animation indicating LPU inference processing. Chevrons and logos are hidden.
<div class="t-pill t-pill--loading">
  <span class="pill-text">
    Converting<span class="pill-dots"></span>
  </span>
</div>
4. Done State t-pill--done
Undo
Highlights green with done styling. Clicking the pill rolls back text adjustments back to original inputs.
<div class="t-pill t-pill--done">
  <span class="pill-text">Undo</span>
</div>
5. Error / Retry State t-pill--error
Retry
Highlights red. Triggered on connection loss or worker proxy failures, allowing quick request re-submits.
<div class="t-pill t-pill--error">
  <div class="pill-icon"><svg>...</svg></div>
  <span class="pill-text">Retry</span>
</div>
